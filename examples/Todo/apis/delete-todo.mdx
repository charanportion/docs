## Step 4: Add Database Node

Add a **Database Node** to delete the todo from the database.

Configure:

- **Database:** Platform DB or MongoDB
- **Collection:** `todos`
- **Operation:** `deleteOne`
- **Filter:**

```
{
  "_id": "{{body.id}}"
}
```

This removes the matching todo document.
